TALK ABOUT

이디야커피 웹사이트를 클로닝하면서 가장 힘들었던 것은 CSS로 반응형 웹사이트를 구현하는 작업이었습니다. PC용 페이지를 디자인 할 때 제대로 구조를 짜놔야 반응형으로 디자인할 때 한 결 수월 하겠다고 깨달았고, swiper slider 기능도 모바일로 가져와서 구현 시키는데에도 사이즈 조절 뿐 아니라 반응형에 맞는 기능코드를 더 넣어야 오류없이 작동하는 것을 보고 웹디자인의 세계는 무궁무진하다는 생각을 했습니다. 뿐만 아니라 Filter 기능을 사용하면서 여러 사이트를 이용하면서 당연하다고 생각하던 기능들이 웹디자이너에게는 결코 쉬운일은 아니라는 것도 깨닫게 되어 좀 더 웹페이지를 보는 안목이 넓 어지는 계기가 되었습니다.

FUNCTION

  • 반응형 웹사이트

    CSS Media를 사용하여 반응형 웹사이트로 Mobile로도 구현 가능하게 함.

  • Modal Detail

    Modal 창을 이용하여 음료의 세부 정보를 한눈에 파악할 수 있게 함.

  • Swiper slider

    Swiper slider를 사용하여 배너와 음료 카테고리에 자동 슬라이드

  • Filter

    Jquery를 이용한 필터 기능으로 음료를 분류하여 검색가능하게 함.
